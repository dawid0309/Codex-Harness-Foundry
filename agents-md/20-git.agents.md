# Git Rules

- Stable branch: `main`
- Milestone branch pattern: `codex/milestone/<m-id>-<slug>`
- Experiment branch pattern: `codex/exp/<topic>-<variant>`
- Default commit cadence: one verified commit per completed task card
- Do not run multiple writers in the same subsystem on the same branch at the same time
